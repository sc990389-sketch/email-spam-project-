# 📧 Email Spam Detection using Machine Learning

A Machine Learning-powered web application that classifies emails as **Spam** or **Ham (Not Spam)** using Natural Language Processing (NLP) and a trained classification model.

## 🚀 Features

* Spam Email Detection
* Real-Time Predictions
* Machine Learning Classification
* Interactive Streamlit Interface
* Fast and User-Friendly

## 🛠️ Tech Stack

* Python
* Streamlit
* Pandas
* NumPy
* Scikit-learn
* Pickle

## 📊 How It Works

1. Enter email text into the input box.
2. The text is converted into numerical features using a trained vectorizer.
3. The Machine Learning model analyzes the input.
4. The application predicts whether the email is Spam or Ham.
5. Results are displayed instantly.

## 🎯 Project Objective

The goal of this project is to identify unwanted spam emails automatically using Machine Learning techniques and text classification.
